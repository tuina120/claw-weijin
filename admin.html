Cloudflare Tunnel/Access（部署提示）
推荐：主站用 Access 严格放行，访客入口单独放行再由体验码控制。
你可以按 README 的步骤配置 Cloudflare Tunnel。Access 推荐做 2 个 Application： 1) claw.qxyx.net 的主站：只允许你的邮箱； 2) 访客入口：/guest* 或 guest.qxyx.net 设为 Bypass（让体验码接管）。
